# Pinyin and dioghv (dieghv) in Rime

拼音和潮语

dioghv in gekio == dieghv in dieziu

gekio 揭阳

dieziu 潮州

## IME

* Fcitx: $HOME/.config/fcitx/rime/
* Ibus: $HOME/.config/ibus/rime/
