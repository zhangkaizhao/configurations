pkg-install.txt:

  History of packages installation.

pkg-query-non-automatic.txt:

  Packages installed manually.
  Query using `pkg query -e '%a = 0' %o`, see `pkg help query`.
  And then `| cut -d'/' -f2` to select only package name.
  That is `pkg query -e '%a = 0' %o| cut -d'/' -f2`.

pkg-absent.txt:

  Packages marked expired or not available yet.
