# Node.js packages

Install with `npm install -g <package>`. See `npm help install`.

* djot

  install:

    npm install -g @djot/djot

* git-stats

  description: Local git statistics including GitHub-like contributions calendars.

  repository: https://github.com/IonicaBizau/git-stats

  install:

    npm install -g git-stats

* katex

  desription: Fast math typesetting for the web.

  homepage: https://katex.org/

  repository: https://github.com/KaTeX/KaTeX

  install:

    npm install -g katex

* pyright:

  install:

    npm install -g pyright

* typescript-language-server

  install:

    npm install -g typescript-language-server

* typescript

  install:

    npm install -g typescript

* yarn

  install:

    npm install -g yarn
