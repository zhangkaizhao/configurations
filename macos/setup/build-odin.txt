# Odin

2026-03-31

version: dev-2026-03

Reference: https://odin-lang.org/docs/install/

The latest version dev-2026-03 requires LLVM. The supported versions of LLVM are 14, 17, 18, 19, 20, and 21.

Install it:

```
sudo port install llvm-21
sudo port select --set llvm mp-llvm-21
```

Build:

```
make release-native
cd vendor/
../misc/remove_libraries_for_other_platforms.sh
cd ..
mkdir /opt/programs/odin
cp -r LICENSE base core examples odin shared vendor /opt/programs/odin/
```

Make a soft link for `libLLVM.dylib`:

```
sudo ln -s /opt/local/libexec/llvm-21/lib/libLLVM.dylib /usr/local/lib/libLLVM.dylib
```

Clean all before update:

```
rm -r /opt/programs/odin/*
```
