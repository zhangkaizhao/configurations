# Python packages

Install with `pip install [options] <package>`. See `pip help install`.

* asciidoc-py

  description: AsciiDoc is a text document format for writing things.

  repository: https://github.com/asciidoc-py/asciidoc-py

  install:

      pip install asciidoc

* cohesion

  description: Cohesion is a tool for measuring Python class cohesion.

  repository: https://github.com/mschwager/cohesion

  install:

      pip install cohesion

  note: Development of Cohesion has been suspended since 2019. No longer in use.

  This package is used as a flake8 extension.

* dlint

  description: Dlint is a tool for encouraging best coding practices and helping ensure Python code is secure.

  repository: https://github.com/duo-labs/dlint

  install

      pip install dlint

  note: Development of Dlint has been suspended since 2020. No longer in use.

  This package is used as a flake8 extension.

* docutils

  description: Docutils -- Python Documentation Utilities

  homepage: http://docutils.sourceforge.net/

  console scripts:

    - docutils
    - rst2html
    - rst2html4
    - rst2html5
    - rst2latex
    - rst2man
    - rst2odt
    - rst2pseudoxml
    - rst2s5
    - rst2xetex
    - rst2xml

  install:

      pip install docutils

* flake8

  description: the modular source code checker: pep8 pyflakes and co

  repository: https://github.com/PyCQA/flake8

  install:

      pip install flake8

* ipython

  description: IPython: Productive Interactive Computing

  repository: https://github.com/ipython/ipython

  install:

      pip install ipython

* notebook

  description: A web-based notebook environment for interactive computing

  repository: https://github.com/jupyter/notebook

  install:

      pip install notebook

  note: run `python -m notebook` to start

* pip-audit

  description: A tool for scanning Python environments for known vulnerabilities

  repository: https://github.com/trailofbits/pip-audit

  install:

      pip install pip-audit

* pip-tools

  description: pip-tools keeps your pinned dependencies fresh.

  console scripts:

    - pip-compile
    - pip-sync

  repository: https://github.com/jazzband/pip-tools

  install:

      pip install pip-tools

* python-lsp-server

  description: Python Language Server for the Language Server Protocol

  repository: https://github.com/python-lsp/python-lsp-server

  install:

      pip install python-lsp-server

* radon

  description: Code Metrics in Python

  console scripts:

    - radon cc

  repository: https://github.com/rubik/radon

  install:

      pip install radon

* rst2pdf

  repository: https://github.com/rst2pdf/rst2pdf

  install:

  install:

  repository: https://github.com/pypa/virtualenv
