# Fedora

Files:

* initramfs-rescue.img
* initramfs.img
* vmlinuz
* vmlinuz-rescue

For latest kernel of Fedora:

* `vmlinuz`: copied from the latest vmlinuz of Fedora
* `initramfs.img`: copied from the latest initramfs.img of Fedora

For rescue kernel of Fedora:

* `vmlinuz-rescue`: copied from the rescue vmlinuz of Fedora
* `initramfs-rescue.img`: copied from the rescue initramfs.img of Fedora

The latest kernel `vmlinuz` and `initramfs.img` files need to be copied after kernel upgrade each time.
