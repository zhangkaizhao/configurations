# Syslinux

Files:

* ldlinux.e64
* libutil.c32
* menu.c32
* syslinux.cfg
* syslinux.efi

The latest version of Syslinux 6.04-pre1 [1] is downloaded from:

https://www.kernel.org/pub/linux/utils/boot/syslinux/Testing/

The files are copied from:

* `ldlinux.e64`: syslinux-6.04-pre1/efi64/com32/elflink/ldlinux/ldlinux.e64
* `libutil.c32`: syslinux-6.04-pre1/efi64/com32/libutil/libutil.c32
* `menu.c32`: syslinux-6.04-pre1/efi64/com32/menu/menu.c32
* `syslinux.efi`: syslinux-6.04-pre1/efi64/efi/syslinux.efi

The configuration file `syslinux.cfg` is written referred to:

https://wiki.syslinux.org/wiki/index.php?title=Menu

https://wiki.archlinux.org/index.php/Syslinux

https://docs.slackware.com/howtos:slackware_admin:set_up_syslinux_as_boot_loader_on_uefi_based_hardware


[1]: https://www.kernel.org/pub/linux/utils/boot/syslinux/Testing/6.04/syslinux-6.04-pre1.zip
