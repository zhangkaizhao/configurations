# Ruby gems

Install with `gem install [options] GEMNAME`. See `gem help install`.

* asciidoctor

  description: A fast, open source text processor and publishing toolchain for converting AsciiDoc content to HTML 5, DocBook 5, and other formats.

  repository: https://github.com/asciidoctor/asciidoctor

  install:

      gem install asciidoctor

* debug

  description: Debugging functionality for Ruby.

  repository: https://github.com/ruby/debug

  install:

      gem install debug

  for VSCode Ruby rdbg Debugger https://github.com/ruby/vscode-rdbg

* jekyll

  description: Jekyll is a simple, blog aware, static site generator.

  repository: https://github.com/jekyll/jekyll

  install

      gem install jekyll

* jekyll-import

  description: Provides the Import command for Jekyll.

  repository: https://github.com/jekyll/jekyll-import

  install

      gem install jekyll-import

* pry

  description: A runtime developer console and IRB alternative with powerful introspection capabilities.

  repository: https://github.com/pry/pry

  install:

      gem install pry

* rubocop

  description: a Ruby code style checking and code formatting tool

  repository: https://github.com/rubocop/rubocop

  install:

      gem install rubocop

* rucoa

  description: Language server for Ruby.

  repository: https://github.com/r7kamura/rucoa

  install:

      gem install rucoa

  for VSCode extension vscode-rucoa https://github.com/r7kamura/vscode-rucoa

* solargraph

  description: IDE tools for code completion, inline documentation, and static analysis

  repository: https://github.com/castwide/solargraph

  install:

      gem install solargraph

* standard

  description: Ruby Style Guide, with linter & automatic code fixer

  repository: https://github.com/testdouble/standard

  install:

      gem install standard

* textbringer

  description: An Emacs-like text editor

  repository: https://github.com/shugo/textbringer

  install:

      gem install textbringer

* typeprof

  description: A type analysis tool for Ruby code based on abstract interpretation

  repository: https://github.com/ruby/typeprof

  install:

      gem install typeprof
